# giacomoanastasia.github.io

Personal academic website of Giacomo Anastasia, PhD student in Economics at Columbia University.

Built with [Hugo](https://gohugo.io) and hosted on [GitHub Pages](https://pages.github.com). Based on the [Hugo academic website template](https://github.com/pmichaillat/hugo-website) by Pascal Michaillat, which uses the [PaperMod theme](https://github.com/adityatelange/hugo-PaperMod) by Aditya Telange.
